Frequently Asked Questions

Why is nothing showing in the text preview pane?
There are a couple of possibilities:
  • You have not yet input a PDF, or have not processed the input PDF yet. See Extracting text from a PDF for instructions.
  • The input PDF does not have any textual data in it. Some PDFs may look like they contain text, but actually only have bitmap imags or vector representations of text. PDFtoText X cannot extract this kind of data; you will need to find software capable of optical character recognition (OCR) in order to convert these kinds of data to text.
