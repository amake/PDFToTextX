PDFtoText X Help

Extracting text from a PDF

Extracting

PDFtoText X allows you to easily extract all textual data from one or more PDFs.

To begin, simply input one or more PDFs in any of the following ways:

  • Select "Open..." from the File menu.
  • Click the "Select a file..." button on the main window.
  • Drag-and-drop PDF(s) onto the main window.
  • Drag-and-drop PDF(s) onto the application icon in the Finder or in the Dock.

If "Refresh text preview automatically" is enabled, the extracted text will automatically appear in the right-hand side pane. If not, click "Go" to begin extracting. Extracting is usually very fast, though in some cases it can take a few seconds (especially when extracting from large PDFs accessed over slower network connections).

See Advanced Settings for more information about extraction options.

Once the text has been extracted you can edit it in the preview pane before saving it.

Saving

If "Automatically save results" was checked when you began extraction, the extracted text file is already in the specified destination folder. If your original PDF was named MyDocument.pdf, then the extracted text file will be MyDocument.txt.

If the results were not automatically saved, or if you have made changes to the text in the preview pane, save the results by clicking the "Save..." button, or click "Copy" to copy the text to the clipboard.
