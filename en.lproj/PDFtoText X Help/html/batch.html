Batch processing

By inputting multiple PDFs at once, you can batch extract text from all of them in one operation.

Make sure the following settings are enabled before beginning batch extraction:

  • Check "Refresh text preview automatically".
  • Check "Automatically save results in:". Set a custom destination if desired.
  • To maximize extraction speed, uncheck "Display PDF".

Then, simply drag-and-drop multiple PDFs onto the PDFtoText X main window or application icon.
