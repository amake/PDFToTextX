PDFtoText X ヘルプ

PDF からテキストを抽出する

抽出

PDFtoText X で、 PDF ファイルに収録されているすべてのテキストデータを簡単に抽出・保存できます。

まずは以下のいずれかの手段で PDF ファイルを入力します。

  • ファイルメニューの「開く...」を選択する
  • メインウィンドウの「ファイルを選択...」ボタンをクリックする
  • PDF ファイルをメインウィンドウにドラッグ＆ドロップする
  • PDF ファイルを、Finder まはた Dock 内のアプリケーションアイコンにドラッグ＆ドロップする

「テキストプレビューを自動更新」が入になっている場合、抽出されたテキストは自動的にウィンドウの右側にある表示枠に表示されます。同設定が切の場合は「実行」ボタンをクリックすると処理が開始されます。処理は普段速いですが、低速なネットワーク接続を通して大容量の PDF ファイルにアクセスしている場合などは数秒かかることがあります。

抽出の設定については詳細設定をご参照ください。

抽出されたテキストはプレビューの表示枠内で編集し保存することができます。

保存

抽出開始時に「抽出したテキストを自動的に以下の場所に保存する」が入になっていた場合、抽出テキストはすでに指定フォルダ内に保存されています。例えばもとの PDF ファイル名が MyDocument.pdf の場合、抽出テキストは MyDocument.txt という名称で保存されます。

抽出テキストが自動的に保存されていない場合、または抽出後テキストプレビューで編集を行なった場合、「保存...」ボタンをクリックして保存するか、「コピー」ボタンをクリックしてクリップボードにコピーします。
