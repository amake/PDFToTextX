PDFtoText X ヘルプ

詳細設定

PDFtoText X は pdftotext というコマンドラインツールを UI 化したものです。 PDFtoText X の詳細設定のほとんどは、 pdftotext をターミナルなどで実行するときに指定できる引数に相当します。

詳細設定はメインウィンドウと環境設定で設定できます。

メインウィンドウ

以下の説明文は pdftotext の manpage 内容を翻訳したものです。

名前 説明 コマンドライン
もとのレイアウトを模倣 PDF のとおりのテキスト配置をなるべく正確に再現する。逆にデフォルトではテキストの位置関係やハイフン処理を排除して素のテキストを出力する。 -layout
改ページを挿入 ページの間に改ページ（form feed 文字）を挿入しない（チェック解除時）。 -nopgbrk （解除時）
Raw output Keep the text in content stream order. This is a hack which often "undoes" column formatting, etc. Use of raw mode is no longer recommended. -raw
ページ 出力する最初のページを指定する。 -f
出力する最後のページを指定する。 -t
改行文字 改行文字の種類を設定する。（デフォルトは「unix」） -eol
テキストエンコーディング 出力のテキストエンコーディングを設定する。（デフォルトは「UTF-8」） -enc
パスワードを使用　作成者 PDF の作成者パスワードを指定し、すべての制限を解除する。 -opw
パスワードを使用　閲覧者 PDF の閲覧者パスワードを指定する。 -upw

「初期値に戻す」ボタンをクリックして以上のすべての設定をデフォルト値に戻します。

環境設定ウィンドウ

「pdftotext バイナリを指定」すると、本アプリケーションが提供しているバージョン以外の pdftotext を代用できます。ただし、指定したバイナリが同様の引数に対応していない場合は誤動作する可能性があります。
